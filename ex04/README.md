#  第4回
##  逃げろこうかとん (ex04/dodge_bomb.py)
###  ゲーム概要
-  ex04/dodge_bomb.pyを実行すると、1600x900のスクリーンに草原が描画され、こうかとんをキーボードで移動させ爆弾から逃げるゲーム
-  こうかとんが爆弾に接触するとゲームオーバーとなりゲームが終了する
###  操作方法
-  矢印キーでこうかとんを上下左右に移動させる
###  追加機能
-  壁に当たると爆弾の色が変化する
-  ゲームが終了すると終了したという報告がされる
